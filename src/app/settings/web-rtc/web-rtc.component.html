Unprotected
WebRTC has the right to enumerate all interfaces and bind them to discover public interfaces.
Default Public Interface Only
Sends WebRTC traffic via the default public network adapter. This will be the VPN adapter for system-VPN users. WebRTC should only use the default route used by http.
Default Public And Private Interfaces
Same as Default Public Interface Only, except allowing WebRTC traffic through the default private interface to your local network as well.
Disable Non Proxied UDP
Disables non-proxied UDP and forces proxy. This policy forces the use of a proxy, and only allows WebRTC traffic over UDP proxies. This will effectively disable WebRTC communication for most users (depending on UDP proxy usage).
